# nselookup
Keyword search for nmap NSE scripts displaying the readme for each.

./nselookup - List nmap NSE scripts based on a keyword in their filename

USAGE
```
./nselookup -k keyword
```

WHERE	keyword   - a word to check nmap NSE script titles

EXAMPLE	
```
./nselookup -k vnc
```
